## [2026-03-31] 인사이트 기능 1차 개발

### 개요
사용자가 기록을 **습관화**하고 기록에서 **실제 인사이트**를 얻도록 하는 기능 4종을 앱/웹 동시 구현.

### 구현 기능

#### 1. 연속 기록 스트릭
- 홈 화면에 🔥 N일 연속 기록 배너 표시
- 오늘 또는 어제 기록이 있으면 연속일 계산 시작
- 기록 없는 날 스트릭 초기화

#### 2. 미완료 nextAction 배너
- 홈 화면에 📌 "아직 못 한 것들 N개" 배너 표시
- 탭/클릭하면 기록 목록의 미완료 탭으로 이동

#### 3. 주간 요약 카드
- 홈 화면에 지난 7일 총 기록 / 완료율 / 가장 많이 반복된 실수 유형 표시

#### 4. 미완료 필터 탭 (기록 목록)
- 전체 / 미완료 탭 전환
- 미완료 없을 때 🎉 빈 상태 메시지

#### 5. 감정 강도 트렌드 차트 (리포트)
- 일별 평균 강도를 바 차트로 시각화
- 낮음(민트) / 보통(노랑) / 높음(빨강) 색상 구분
- 7일: 전체 표시, 30일: 5일 간격으로 표시

## [2026-04-01] 라이트/다크 테마 시스템 구축

### 개요
웹과 모바일 앱 모두 라이트 모드 / 다크 모드 전환 기능을 구현. 사용자가 각 플랫폼에서 자유롭게 테마를 전환 가능.

### 웹 변경 사항

#### CSS 변수 기반 시맨틱 토큰 시스템
- `tailwind.config.js`: 하드코딩 색상 → CSS 변수 참조 토큰으로 전환
  - `bg-page`, `bg-card`, `bg-elevated`, `bg-overlay`, `border`, `text-primary`, `text-secondary`, `text-muted`
  - `rgb(var(--token) / <alpha-value>)` 형식 — Tailwind opacity modifier 완전 지원
- `apps/web/src/index.css`: `:root`(라이트) + `.dark`(다크) CSS 변수 값 정의
  - 라이트: BG `#F4F6FB`, CARD `#FFFFFF`, 텍스트 `#1A2030`
  - 다크: BG `#202631`, CARD `#2B3340`, 텍스트 `#EAF0FA`
- `apps/web/index.html`: FOUC 방지 인라인 스크립트 추가 (React 렌더 전 `.dark` 클래스 적용)

- `Layout.tsx`: `useTheme()` 훅 → 사이드바 하단에 ☀️/🌙 토글 버튼 추가
